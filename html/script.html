PowerShell Script Examples

Find Files in a Directory

# List all files in the specified directory Get-ChildItem -Path C:\Path\To\Directory -File

Search for Content in Files

# Search for a specific string in text files within a directory Get-ChildItem -Path C:\Path\To\Directory -Filter *.txt | ForEach-Object { if (Get-Content $_.FullName | Select-String -Pattern "SearchText") { $_.FullName } }

Move Files to Another Directory

# Move files from one directory to another Get-ChildItem -Path C:\Source\* -File | Move-Item -Destination C:\Destination\

List all files with a specific file extension:

Get-ChildItem -Path C:\Path\To\Directory -Filter *.txt

Recursively list all files in subdirectories:

Get-ChildItem -Path C:\Path\To\Directory -Recurse

Search for files containing a specific string:

Get-ChildItem -Path C:\Path\To\Directory -Recurse | Select-String -Pattern "searchString"

Find the newest or oldest file in a directory

# Find the newest file Get-ChildItem -Path C:\Path\To\Directory | Sort-Object LastWriteTime -Descending | Select-Object -First 1 # Find the oldest file Get-ChildItem -Path C:\Path\To\Directory | Sort-Object LastWriteTime | Select-Object -First 1

Search for files based on size:

# Find files larger than a specific size (e.g., 1GB) Get-ChildItem -Path C:\Path\To\Directory | Where-Object { $_.Length -gt 1GB } # Find files smaller than a specific size (e.g., 100MB) Get-ChildItem -Path C:\Path\To\Directory | Where-Object { $_.Length -lt 100MB }

Conditional statements in PowerShell

$number = 10 if ($number -lt 5) { Write-Host "The number is less than 5." } elseif ($number -eq 10) { Write-Host "The number is exactly 10." } else { Write-Host "The number is neither less than 5 nor exactly 10." }

comparing String in power shell script

$input = "Hello" if ($input -eq "Hello") { Write-Host "The input is Hello." } elseif ($input -like "He*") { Write-Host "The input starts with He." } else { Write-Host "The input does not match." }

Using ForEach-Object Cmdlet

$numbers = 1..5 $numbers | ForEach-Object { Write-Host "Number: $_" }
